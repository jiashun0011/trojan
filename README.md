# prerequisites
- VM
  - [Azure](https://azure.microsoft.com/)
  - [HOSTKVM](https://hostkvm.com/)
- Domain
  - [GoDaddy](https://www.godaddy.com/)
- Client tool
  - [Maple](https://github.com/YtFlow/Maple)
  - [Mellow](https://github.com/mellow-io/mellow)(No longer supported)

# install 
```bash
$ ./install_trojan.sh
```

# debug
```bash
$ journalctl -u trojan
```
